# e-Commerce de Adopción de Mascotas

## Descripción
Este proyecto es una página web básica desarrollada para un curso de Front-End.  
El sitio muestra información sobre compra en un criadero de perros, incluye una sección de productos cachoros, un formulario de contacto y navegación entre páginas internas.

## Estructura
- index.html: Página principal, con un baner, navegación, logo, imagen y lista de perros para comprar o reservar organizada en tarjeta.
- Conocenos.html: Información de la organización.
- Contactanos.html: Formulario de contacto y tabla ficticia de mensajes escibidos.
- styles.css → Estilos globales para todas las páginas.
